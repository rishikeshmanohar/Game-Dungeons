
# Game Dungeon

[**[Play Game!]**](https://game-dungeons.vercel.app/)

Game Dungeon is a fun 2D platformer game where you explore dungeons, jump over obstacles, and fight enemies. I'm making this game to learn Unity and get better at game development.

## How to Play

**Controls:**
- **A / Left Arrow** - Move left
- **D / Right Arrow** - Move right
- **Space / W / Up Arrow** - Jump
- **Click** - Interact with doors and buttons

**Game Goals:**
1. Use the arrow keys or A/D to move your character
2. Jump over spikes and obstacles to avoid taking damage
3. Defeat enemies by jumping on them
4. Collect coins and gems for points
5. Find the red door at the end of each level
6. Reach the door to complete the level


## Setup Instructions

Follow these steps to set up and play the game on your computer or Mac.

### Step 1: Download and Install Unity

1. Go to [Unity Download](https://unity.com/download)
2. Download **Unity 6** (version 6000.0.32f1 or newer)
3. Follow the installer and complete the installation
4. Create a free Unity account if you don't have one

### Step 2: Download the Project

**Option A: Using Git**
1. Open your terminal or command prompt
2. Go to the folder where you want to save the project
3. Run this command:
   ```
   git clone https://github.com/rishikeshmanohar/Game-Dungeons.git
   ```
4. Wait for the download to finish

**Option B: Download as ZIP**
1. Go to [Game-Dungeons Repository](https://github.com/rishikeshmanohar/Game-Dungeons)
2. Click the green **Code** button
3. Click **Download ZIP**
4. Extract the ZIP file to your desired location

### Step 3: Open the Project in Unity

1. Open Unity Hub
2. Click **Open** or **Add project from disk**
3. Navigate to the Game-Dungeons folder and select it
4. Wait for Unity to load the project (this may take a few minutes)

### Step 4: Play the Game

1. In the Unity Editor, go to **Assets > _Scenes** folder
2. Open **__Main.unity** scene (this is the main menu)
3. Click the **Play** button at the top of the editor
4. Enjoy the game!

### Step 5: Build the Game (Optional)

If you want to create a playable version:

1. Go to **File > Build Settings**
2. Make sure the scenes are added to the build
3. Choose your platform (Windows, Mac, WebGL, etc.)
4. Click **Build** and select where to save it
5. Run the built game file

**Note:** Some features may only work correctly with Unity 6 or newer.
